# Orange ERP

## Vision

Orange ERP is a cloud-based ERP platform designed for Internet Service Providers (ISPs).

The first deployment target is Orange Fibernet.

---

## Technology Stack

- Backend: NestJS
- Frontend: React + Vite
- Database: PostgreSQL
- ORM: Prisma
- Cache: Redis
- Reverse Proxy: Nginx
- Deployment: Docker Compose
